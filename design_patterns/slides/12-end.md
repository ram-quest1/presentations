### What We Covered

| Pattern | Key Idea |
|---------|----------|
| Singleton | One instance, globally accessible |
| Factory | Centralized object creation + pooling |
| Decorator | Add behavior by wrapping |
| Proxy | Control access to an object |
| Observer | Notify many when one changes |
| Visitor | Add operations without modifying classes |
